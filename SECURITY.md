# Security Policy

## ⚠️ Important notice

This repository contains **intentionally vulnerable code** for educational
and demonstration purposes. The `app/` directory is a deliberately insecure
Python application designed to showcase AI-powered security tooling.

**Never deploy any code from this repository in a real environment.**

## Scope

This security policy covers the repo infrastructure (GitHub Actions workflows,
hook scripts, MCP configuration) — NOT the vulnerable app, which is
intentionally insecure by design.

## Reporting a vulnerability

If you find a security issue in the repo infrastructure itself, please open
a private vulnerability report:

GitHub → Security tab → Report a vulnerability

## Supported versions

This is a portfolio and educational project. No versioned releases are maintained.
